# 브랜드/네이밍 정의서 v1.0

홈페이지 개발/디자인/콘텐츠 제작 시 **표기 혼선 방지**를 위한 SSOT(단일 진실 원천) 문서

* 작성일: 2025-12-28 (Asia/Seoul)
* 적용 범위: 홈페이지(타이틀/푸터/OG), 네이버 플레이스, 인스타/블로그, 명함/간판, PDF/안내문
* 원칙: **한 번 정한 표기는 모든 채널에서 동일하게 반복**한다.

---

## 0) 공통 표기 원칙

### 0.1 대소문자/띄어쓰기

* 영문 브랜드명은 **Title Case** 고정

  * 예: `Wonder Pilates`, `K Classic Pilates`, `K Classic Pilates Association`
* 약어는 **전체 대문자** 고정

  * 예: `KCPA`
* 한글 브랜드명은 **띄어쓰기 고정**

  * 예: `경이로운 필라테스`, `K클래식 필라테스`

### 0.2 한/영 병기 규칙

* 기본 병기 포맷(권장): `한글명 | English Name`

  * 예: `경이로운 필라테스 | Wonder Pilates`
* 영어권 대상 문서에서는 영문을 먼저: `Wonder Pilates | 경이로운 필라테스`

### 0.3 금지 규칙(혼선 방지)

* `WONDER PILATES`(올캡스) 메인 사용 금지(가독성/톤 일관성 저하)
* `K-Classic`(하이픈) 메인 사용 지양(외부 유사 표기와 혼동 가능)
* 한글/영문을 페이지마다 바꾸는 표기 금지(채널별 표기 불일치 금지)

  * `Wonder Pilates` 단독
